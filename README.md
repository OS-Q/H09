### [简介](https://github.com/OS-Q/P41/wiki)

[P41](https://github.com/OS-Q/P41) 适用于 [nRF52](https://www.nordicsemi.com/Products/Low-power-short-range-wireless/Bluetooth-low-energy) 平台开发

#### 平台特性

* BLE 5.0
* NFC
* Arduino

定位移动端交互平台，用于和移动设备进行BLE/NFC数据交互的直连。
